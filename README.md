# 🚀 SaaS Dashboard (Next.js + Redux Toolkit)

## 📌 Overview
A full-featured dashboard built with Next.js, Redux Toolkit, and Material UI.

## ✨ Features
- 🔐 Authentication (Login + Protected Routes)
- 👥 Users Management
- 📦 Products with Cart System
- 📝 Posts + Comments System
- 📊 Analytics Dashboard (Charts)
- 🌙 Dark / Light Mode

## 🧠 Tech Stack
- Next.js (App Router)
- TypeScript
- Redux Toolkit
- Material UI
- Recharts
- DummyJSON API
